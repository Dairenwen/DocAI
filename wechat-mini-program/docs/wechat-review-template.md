# 小程序提审说明模板

更新时间：2026-04-05

## 一、产品简介

`DocAI` 是一款面向文档处理与智能填表场景的小程序，提供资料上传、文档解析、AI 问答、模板填表、结果下载与账号管理能力，帮助用户在小程序内完成资料整理与结果输出。

## 二、核心功能

1. 账号登录与注册
2. 文档上传与解析
3. 基于文档的 AI 问答
4. 智能填表与结果生成
5. 结果下载与转发
6. 账号安全、联系我们、协议与隐私入口

## 四、测试步骤

1. 打开小程序，勾选《用户协议》《隐私政策》后登录。
2. 进入“文档”页，上传一份 `docx / xlsx / md / txt` 资料。
3. 等待文档状态变为可用，进入“对话”页发起 AI 问答。
4. 进入“智能填表”页，选择数据源与模板文件，填写需求并开始执行。
5. 执行完成后下载结果文件，或尝试转发结果文件。
6. 进入“我的”页，可查看协议、联系我们、账号与安全入口。

## 六、隐私说明

- 小程序已提供《用户协议》《隐私政策》本地页面。
- 登录、上传资料等涉及个人信息处理的操作已接入隐私授权流程。
- 当前处理的信息包括：
  - 账号信息
  - 用户主动上传的文件信息
  - AI 问答内容
  - 智能填表需求与结果
  - 设备与运行日志信息
- 用户可通过“账号与安全”入口直接注销当前账号；“联系我们”入口用于问题反馈和服务咨询。
